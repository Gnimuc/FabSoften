For clueless Visual Sudio users

  1. Find FabSoften.sln in the build folder -> double-click FabSoften.sln to open the solution file in Visual Studio
  2. Find Solution Configurations(defaults to Debug x64) and change Debug to Release:
    • left-click Debug to open a dropdown menu
    • left-click Release in the menu
  3. Find the project named FaceLandmarkDetection in the Solution Explorer
    • right-click FaceLandmarkDetection
    • left-click Build in the prompt menu
    • right-click FaceLandmarkDetection again
    • left-click Open in Terminal in the prompt menu
  4. Copy the code in the following code block to the Develop Powershell:
    .\Release\FaceLandmarkDetection.exe ..\..\..\models\shape_predictor_68_face_landmarks.dat ..\..\..\assets\pexels-aadil-2598024.jpg
    • Hover your cursor onto the above code block
    • Now a button appears at the right hand side of the code block
    • Move your cursor over that button and left-click the button
  5. Find the Develop Powershell window and hover your cursor on the window, then paste the code by doing a right-click
  6. Press enter
